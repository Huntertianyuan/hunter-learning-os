# Effective Feedback Template

```text
Topic:

Knowledge Gain:
- 

Cognitive Gain:
- 

Worth Saving:
- Memory Type:
- Save Target:
- Why it matters:

Review Queue:
- Recall Prompt:
- Recall Type:
- Recall Strength:
- Suggested Interval:

Future Use:
- 
```

Use this at meaningful stopping points.

Do not force Cognitive Gain.

If there is no visible cognitive shift, write:

```text
Cognitive Gain:
No clear cognitive shift yet.
```
