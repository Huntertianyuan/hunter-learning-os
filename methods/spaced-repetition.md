# Method: Spaced Repetition

## Purpose

Schedule review over increasing intervals to strengthen long-term retention.

This method helps important concepts survive beyond the current session.

## When to Use

- The user wants to remember material over time.
- Active Recall identifies concepts worth reviewing.
- A concept is important but likely to fade.
- The system is designing a review plan.

## When NOT to Use

- The material has not been understood yet.
- The item is not worth long-term retention.
- The user needs application or reflection more than repetition.
- The review burden would become too large.

## Inputs

- Review item or prompt.
- Desired retention horizon.
- Current recall strength.
- Importance level.

## Workflow

1. Convert material into a recallable prompt.
2. Set an initial review interval.
3. At each review, use Active Recall before showing the answer.
4. Increase interval when recall is strong.
5. Shorten interval when recall is weak.
6. Retire or revise items that are no longer useful.

## Outputs

- Review prompts.
- Suggested review schedule.
- Recall performance notes.
- Retired or revised items.

## Success Criteria

- The user remembers important material across time.
- Reviews stay focused and manageable.
- Prompts test understanding, not rote wording only.
- Weak items are repaired rather than endlessly repeated.

## Memory Integration

- Add review candidates to Memory.
- Update Concept History with recall performance.
- Track retention patterns in Learning Profile.
- Link repeated review failures to Misconception History.

## Compatible Skills

- Review
- Deep Dive
