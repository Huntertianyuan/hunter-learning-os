# Method: Interleaving

## Purpose

Mix related concepts, problems, or skills during practice to improve discrimination and transfer.

This method helps the learner recognize which concept or method applies in changing contexts.

## When to Use

- The user confuses similar concepts.
- The user can solve problems in isolation but struggles in mixed situations.
- Multiple related topics need review.
- The goal is flexible application rather than simple repetition.

## When NOT to Use

- The user has not learned the individual concepts yet.
- The mixed practice would become overwhelming.
- The task is first exposure or basic explanation.
- The concepts are unrelated and mixing them would add noise.

## Inputs

- Set of related concepts or problem types.
- Known confusions.
- Practice examples or prompts.
- Current skill level.

## Workflow

1. Select 2-4 related concepts or problem types.
2. Create mixed prompts without labeling the type upfront.
3. Ask the user to identify what kind of concept or method applies.
4. Compare the user's choice with the correct distinction.
5. Explain differences where confusion appears.
6. Repeat with varied examples.

## Outputs

- Mixed practice set.
- Discrimination errors.
- Clarified boundaries.
- Improved transfer readiness.

## Success Criteria

- The user can tell similar concepts apart.
- The user chooses appropriate methods in context.
- Confusion decreases across examples.
- Practice remains challenging but not chaotic.

## Memory Integration

- Update Misconception History with confusion pairs.
- Add mixed review prompts to Memory.
- Track discrimination improvements in Concept History.
- Update Learning Profile with topics needing interleaving.

## Compatible Skills

- Review
- Deep Dive
- Application
